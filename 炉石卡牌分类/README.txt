# 炉石传说卡牌分类目录

## 目录结构
- 扩展包(set)：第一级目录，包含all_cards.json文件（该扩展包的所有卡牌）
- 职业(cardClass)：第二级目录，包含all_cards.json文件（该职业的所有卡牌）
- 稀有度(rarity)：第三级目录，包含cards.json文件（该稀有度的所有卡牌）

每个目录都有一个JSON文件，包含该层级的所有可收藏卡牌的完整信息。
